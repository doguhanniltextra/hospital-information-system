# Security Policy

## Project Context

This repository contains a portfolio project demonstrating a microservices-based Hospital Information System. It is not currently deployed in a live production environment handling real patient data.

## Supported Versions

Only the latest commit on the `main` branch is actively maintained.

## Reporting a Vulnerability

Even though this is an educational/portfolio project, I adhere to standard security disclosure practices.

If you review this architecture or codebase and discover a security flaw—particularly regarding edge security (API Gateway), JWT handling, or tenant data isolation—please do not open a public issue.

Submit a report detailing the vulnerability and a proof-of-concept directly to Doğuhan İlter at doguhannilt@gmail.com.

I will review the report, implement a patch, and credit you in the project documentation for the finding.
